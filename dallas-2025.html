What makes a Web Component?

  • Custom Elements
  • Shadow DOM
  • HTML Templates

Custom Elements

  • must include a hypen/dash (kebab-case), no single words
  • a class object that defines the behavior
  • which built-in elements it extends, if any
Source: MDN

Shadow DOM

  • provides encapsulation
  • "...contain the gross" - Paul Lewis (link to video)
  • remap events. ie, turn a click event into something more meaningful
Source: MDN

HTML Templates

  • < template > element's content is not immediately rendered. We define the HTML structure upfront and clone when needed via JavaScript.
  • used to efficiently render HTML and update it when the user interacts with our web component
  • for complex Templates, may be easier to use a library, framework or tool
Source: MDN
